# Computer Code Availability

## English statement for the manuscript

The computer code used in this study is openly available in a public GitHub repository:

<https://github.com/xiaochou12138/Joint-Recognitiont-Based-on-DINOv2-Foundation-Model-Fine-Tuning>

The repository contains the source files for data preparation, DINOv2-based model fine-tuning, single-task and multi-task training, evaluation, post-processing, and visualization. It also includes a `README.md` file, usage documentation, dependency information, an Apache-2.0 open-source license, and a lightweight quick test that can be run with:

```bash
python tests/quick_test.py
```

The repository can be downloaded anonymously from GitHub or cloned with:

```bash
git clone https://github.com/xiaochou12138/Joint-Recognitiont-Based-on-DINOv2-Foundation-Model-Fine-Tuning.git
```

Large seismic datasets and trained model outputs are not stored as source code in the repository. Data access and placement instructions are provided in the repository documentation.

## 中文稿件说明

本研究所使用的计算机代码已在公共 GitHub 仓库中开源发布：

<https://github.com/xiaochou12138/Joint-Recognitiont-Based-on-DINOv2-Foundation-Model-Fine-Tuning>

该仓库包含数据准备、基于 DINOv2 的模型精细校正、单任务与多任务训练、模型评估、后处理和可视化所需的源代码文件，并包含 `README.md`、使用文档、依赖说明、Apache-2.0 开源许可证以及可快速运行的测试脚本：

```bash
python tests/quick_test.py
```

该仓库支持匿名下载，也可通过如下命令克隆：

```bash
git clone https://github.com/xiaochou12138/Joint-Recognitiont-Based-on-DINOv2-Foundation-Model-Fine-Tuning.git
```

大型地震数据集和训练输出文件未作为源代码存放在仓库中。数据获取与本地放置方式已在仓库文档中说明。
